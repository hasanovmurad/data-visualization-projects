# data-visualization-projects

# Data Visualization Projects

This repository contains two independent data visualization and exploratory data analysis (EDA) projects implemented in Python using Jupyter Notebook.

---

## 1. Cell Specification Analysis

### Objective
Analyze and visualize mobile phone specifications to identify patterns across brands, price segments, and hardware characteristics.

### Key Tasks
- Data cleaning and preprocessing
- Exploratory data analysis (EDA)
- Feature distribution analysis
- Correlation analysis
- Comparative visualizations

### Tools & Libraries
- Python
- pandas
- numpy
- matplotlib
- seaborn

---

## 2. Stock Market Analysis

### Objective
Perform time-series visualization and trend analysis on stock market data.

### Key Tasks
- Time-series preprocessing
- Trend visualization
- Moving averages (if used)
- Volatility analysis (if used)

### Tools & Libraries
- Python
- pandas
- matplotlib
